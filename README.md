#  Flight Search Application

A Java-based flight search application developed as part of the 
Master of Information Technology program at RMIT University.

##  Built With
- Java
- Maven
- Object-Oriented Programming (OOP)

##  Features
- Flight search functionality
- Structured data management
- Modular, maintainable codebase using OOP principles

##  Academic Context
Developed during postgraduate studies at RMIT University, 
Melbourne  demonstrating core software engineering and 
Java development skills.
